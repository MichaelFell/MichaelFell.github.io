Balance Belt

Project Description

This was my first project at Reality Garage - I built the prototype of a belt that provides balance cues to people suffering from vestibular (inner ear) issues. I was tasked with integrating four gyroscopes and haptic units together to measure the user's orientation and help them correct when they could not sense the change themselves.

My device went on to be used for a Summer-long research project in the Aerospace labs at the University of Colorado at Boulder.

Skills Used

  • C# programming in Unity
  • Rapid Prototyping
  • Firmware

Delaunay Mesh Refinement Optimization

Project Description

I took on this project as a paper topic in my senior year Concurrent Programming class. I was tasked with building a working, multi-threaded executor of the Delaunay mesh refinement algorithm, which measures the "quality" of triangles in a mesh and re-topologizes it to optimize the structure. I'll admit it didn't work super well, but it did work.

Skills Used

  • Multi-threading
  • Algorithm writing
  • C++ programming
