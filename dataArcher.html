Data Archer VR

Project Description

Data Archer is a game project built at Reality Garage for Flexential, an IT service management company. They wanted a fun, educational game that they could bring to conferences, and we designed a VR archery trivia-style game where the user had to shoot application-arrows into the correct servers, based on computing needs.

I headed up asset creation and animation for this project - it was my first time designing objects meant for heavy VR interaction, meaning they had to look good, feel good, and operate smoothly.
